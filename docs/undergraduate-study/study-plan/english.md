本科生英语课程是通识教育课程中公共课程类的必修部分。新生入学后须参加学校统一组织的英语分级考试，根据考试结果确定修读级别。

自 2025 级起，学校对本科英语课程体系进行了改革，课程名称与等级划分有所变化，但分级考试、级别调整等核心机制基本延续。

## 分级考试与等级划分

新生入学后统一参加英语分级考试，根据成绩划分等级。2024 级以前与 2025 级以后的英语学分、各等级入学第一学期修读课程名称对比如下：

| | 2024 级（含）以前 | 2025 级（含）以后 |
| :------------: | :------------: | :------------: |
| 累计学分 | 6^[一、二级学生如未免修，或需修读 9 学分] | 4 |
| 一级 | 大学英语（1） | 综合英语Ⅰ |
| 二级 | 大学英语（2） | 综合英语Ⅱ |
| 三级 | 大学英语（3） | 综合英语Ⅲ |
| 四级 | 大学英语（4） | 个性化课程 |

## 级别调整

### 通用规则

入学第一学期第 1–2 周可申请级别调整，适用以下规则：

- 仅限上下调整**一个级别**，不可跨级调整。
- 每人仅可申请**一次**，结果公布后不予更改。
- 每个级别调整人数原则上不超过该级别总人数的 **5%**，内部再分向上一级与向下一级的子限额。
- 向上调整需满足高考英语成绩要求，向下调整无分数要求。
- 留学生、保送生（英语）需提交额外证明材料。
- 需参与级别调整考试，可参考 [2025 级通知](https://jwc.sjtu.edu.cn/info/1222/122771.htm)。

### 高考英语成绩要求

两套体系对应级别的分数要求相同，仅级别名称不同：

| 调整方向 | 高考英语最低要求 |
|----------|-------------------|
| 一级 → 二级 | 130/150 分（含）以上（87% 以上） |
| 二级 → 三级 | 135/150 分（含）以上（90% 以上） |
| 三级 → 四级 | 140/150 分（含）以上（93% 以上） |
| 向下调整（任一方向） | 无分数要求 |

留学生向上调整需提交托福/雅思成绩证明或高中英文教育证明（依原级别而定）；保送生（英语）需提交情况说明。

## 英语免修（2024 级前）

2024 级以前可通过以下两种途径免修后续英语课程；2025 级以后**不设**免修机制，学生只需按分级结果固定修读两个学期。

### 英语水平考试（EPT）

大一春季学期末举行英语水平考试（English Proficiency Test, EPT）。EPT 成绩以一定比例与大一春季学期英语课成绩挂钩，同时决定后续英语课程的免修资格。

经 EPT 考试确定可获得第三学期大学英语免修资格；未获得免修资格的学生须继续修读第三学期《大学英语（3）》或《大学英语（4）》。

### 社会性英语考试成绩免修

也可凭以下社会性英语考试成绩申请免修《大学英语（3）》或《大学英语（4）》：

| 考试类型 | 成绩要求 |
| :- | :- |
| 托福（TOEFL） | 90 分（含）以上 |
| 雅思（IELTS） | 6.5 分（含）以上 |
| 全国大学英语六级（CET-6） | 430 分（含）以上 |

成绩须在提出申请两年之内取得，通过外国语学院审核后方可免修。可参考 [2026 年通知](https://jwc.sjtu.edu.cn/info/1222/125591.htm)。
